通过对13个月大的婴儿的瞳孔测量研究发现：注意力的集中程度可以预测视觉工作记忆的表现

工作记忆 注意
认知心理学
发布日期:   2021-11-30

Focused attention predicts visual working memory performance in 13-month-old infants:a pupillometric study

论文作者：Chen,Cheng,Zsuzsa,Kaldy,Erik,&Blaser.

期刊：Developmental cognitive neuroscience.（2019）

在生命最初的两年里，婴儿积极地与环境进行互动，通过观察和推理迅速学会新的技能，支持这个观点的一个关键机制是视觉工作记忆（VWM），这一机制允许在认知任务中存储和操作视觉信息。

同时，大量证据表明，注意是成年人视觉工作记忆系统的一个关键组成部分。瞳孔测量法是一种敏感，快速，非侵入性的方法，能够用来量化集中注意和VWM之间的关系。有研究发现“任务唤起”的瞳孔反应（在刺激呈现之前或呈现之时的瞳孔直径变化）可以作为注意集中程度的一个指标，同时也可以用它来区分正确和错误的反应以及任务表现的高低。

但是到目前为止，一直缺乏对婴儿的注意集中程度和VWM之间关系的研究，这也就意味着我们对于连接注意和记忆的额叶回路的发展缺乏研究。同时由于在研究中，没有测量婴儿对测试任务相关刺激事件的注意集中程度，我们可能系统地低估了婴儿地VWM能力。因此本研究将通过利用类似于成人瞳孔测量的方法来研究13个月大地婴儿注意集中程度与视觉工作记忆之间的关联。

实验

–实验目的

研究13个月大婴儿注意集中程度与视觉工作记忆表现之间的关系。

–实验设备和刺激

父母将孩子抱在膝盖上，面对Tobii T120眼动仪，要求父母带上面罩遮住他们的眼睛，并且要求在研究期间不与孩子互动。实验刺激是虚拟的“卡片”，可以正面朝上显示一个物体（汽车，球，猫，狗），也可以正面朝下不显示物体。奖励动画：烟花，闪光，闪光灯。

–实验流程

本实验是一个延迟匹配检索任务的变式，每次实验开始时先显示1秒的空屏，之后出现3张不显示物体的卡片持续时间为1.5秒。接着是编码阶段，匹配和不匹配的卡片依次被翻出（匹配和不匹配的卡片始终出现在底部，最上面的卡片式样本卡片），卡片展示时间总长为2.5秒，接着将卡片翻面维持1.5秒的不显示物体的卡片。在反应阶段，将匹配卡片显示在样本卡片的位置，记录此时被试的注视反应。在这个阶段后匹配卡片在原始的位置展现以及伴随着简短的奖励动画。同时一个试次结束后为了吸引被试的注意会出现4秒的卡通太阳。在整个实验的过程中通过眼动仪记录被试的瞳孔变化和注视轨迹。

实验的流程如下图所示：

采用单样本t检验，发现婴儿的平均正确率为59%显著高于随机概率50%。在反应阶段，基于两张牌（匹配或不匹配）中哪一张获得更长的关注时间进行研究发现，匹配卡片的注视总时间显著高于非匹配卡片。

根据婴儿的反应正确率将所有的婴儿分为表现较好和表现较差两组，对比这两组婴儿在编码阶段的TEPR（任务诱发瞳孔反应），发现表现较好组的瞳孔变化量显著高于表现较差组，表明在记忆的编码阶段，较高的TEPR与记忆表现较好的参与者相关。结果如下图所示：

根据记忆反应的正确与否将所有的试次分为正确记忆试次组合不正确记忆试次组，对比这两组在编码阶段的TEPR，发现正确组在编码阶段瞳孔变化量显著高于非正确组。表明在编码过程中，TEPR的大小与记忆反应正确与否相关。结果如下图所示：

总结
本研究通过任务唤起的瞳孔反应（TERP）来捕捉13个月大婴儿的注意力的变化如何影响视觉工作记忆（VWM）的表现。发现注意更加集中的婴儿记忆表现越好。本研究是第一个使用TERP来持续的测量婴儿在VWM中的注意集中程度的研究。但是由于目前选取的样本量较小，未来需要更大的样本量来复制和扩张该项研究。

本文介绍的研究：Chen,Cheng,Zsuzsa,Kaldy,Erik,&Blaser.(2019).Focused attention predicts visual working memory performance in 13-month-old infants:a pupillometric study.Developmental cognitive neuroscience.
